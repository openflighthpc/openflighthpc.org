Flight Solo will eventually be unavailable via the AWS Marketplace however the CloudFormation template and related resources are still available to be manually launched. To access the template and launch Flight Solo, you can:

  • Login to the AWS console
  • Navigate to the CloudFormation app
  • Select 'Create stack'
  • In 'Specify template' set the 'Amazon S3 URL' to https://s3-eu-west-1.amazonaws.com/openflighthpc/templates/solo-cluster.yaml
  • Click 'Next' and update parameters to configure the Flight Solo cluster
