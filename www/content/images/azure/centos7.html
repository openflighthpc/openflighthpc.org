--- heading: OpenFlight Azure CentOS 7 Image Import ---

Prerequisites

  • Azure CLI tool installed
  • Azure CLI tool logged into Azure account
  • A storage container within a resource group with a container for uploading the image to

Importing Image

  • Download image from the OpenFlight repo
  • Upload raw image to a storage container
    • az storage blob upload --account-name mystorageaccount --container-name images --type page --file CENTOS7-OPENFLIGHT-2020.1-1803202029_azure.raw --name CENTOS7-OPENFLIGHT-2020.1-1803202029_azure.vhd
  • Create image from file
    • az image create --resource-group myresourcegroup --name CENTOS7-OPENFLIGHT-2020.1-1803202029_azure --os-type Linux --source https://mystorageaccount.blob.core.windows.net/images/CENTOS7-OPENFLIGHT-2020.1-1803202029_azure.vhd
  • The image is now available in the resource group to create VMs from
