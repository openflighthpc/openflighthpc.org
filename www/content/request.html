Ready to get started in OpenFlightHPC? We’ve got certification and resource packs here.

Welcome to the OpenFlightHPC request site for resources and certification.

In order to ensure our community is made up of the best users for our tools we’ve created a set of certifications to get people started. There is no cost associated, just the ability to get familiar with our core structure in order to help evolve our HPC stack to keep up with the needs of today and tomorrow.

The certifications start with a basic boot camp and run all the way up to a full master certification. You can decide how many levels you want certification in based on your own experience - there is no requirement to do all four levels!

We see the process of cluster creation much like we see the journey up HPC Mountain, each layer building upon the other. As you attain more knowledge with OpenFlightHPC we hope you feed that back to the community in order to make our tool stronger for future generations.

OpenFlightHPC Bootcamp

Starting at base level you will be issued an pre-configured, 8-hour cloud HPC cluster. Using this cluster you will engage in six basic tasks ending in validation and bootcamp certification.

Time Investment: 15 to 30 minutes

Request materials pack to get started now.

Stage 1: Advocate

Using your own pre-defined workload you will learn how to install applications, connect datasets, and learn about workflow script execution. The OpenFlightHPC team will serve as your guides throughout.

Time Investment: 2 weeks, part-time (flexible hours)

Request materials pack to get started now.

Stage 2: Adaptor

This stage focuses on scale, optimisation, and cost analysis, with a focus on completing a short project for publication. At this level our sponsor team at Alces Flight will serve as your guides for certification.

Time Investment: 2-4 weeks, part-time (flexible hours)

Request materials pack to get started now.

Master Certification

The final stage involves creating a workflow sustainability model including porting the OpenFlightHPC stack to different cloud platforms.

Time Investment: 2-4 weeks, part-time (flexible hours)

Request materials pack to get started now.
