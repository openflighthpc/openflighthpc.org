--- heading: Environment ---

This is the third stage in the HPC mountain. Environment involves installing and managing the software, applications and resource usage of the research environment.

Considerations for environment configuration are mainly:

  • Consistent Configuration - Ensuring that version numbers and software settings are the same on all systems that are to run the software.
  • Run Order - Ensuring that the configuration process follows an identical process on nodes to ensure synchronised.
  • Paper Trail - Tracking changes to systems in order to aid troubleshooting.

For more information and guides on environment implementation see the research environment build & management documentation
