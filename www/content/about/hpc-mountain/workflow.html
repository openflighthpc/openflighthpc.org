--- heading: Workflow ---

The final stage of the HPC Mountain. The workflow stage is the combination of the previous stages.

Workflow entails the user processes and application workflows which will be commonplace in the research environment. Generally, the workflow influences the decisions made at earlier stages in the HPC mountain.


For more information on workflow implementation see the research environment build & management documentation and for example workflows with the OpenFlightHPC Compute environment see the environment usage documentation
