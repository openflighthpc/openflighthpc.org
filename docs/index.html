The OpenFlightHPC project is an open-source initiative which provides knowledge and experimental tools regarding High Performance Computing (HPC).

It seeks to cover many aspects of a HPC Service from system architecture all the way to application management and workload life cycles with the ultimate goal of reducing the HPC skills gap.

Our knowledge base and experimental tool sets cover many topics, such as:

  • Service Architecture
  • Green HPC
  • End-user Accessibility
  • Application Management
  • Benchmarking
  • Novel Deployment Methodologies
